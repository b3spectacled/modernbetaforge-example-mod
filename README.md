# Modern Beta for Forge 1.12.2 Example Mod

![Modern Beta Banner](https://i.imgur.com/LFROVPX.png)

An example mod for Modern Beta Forge. This example adds two things:

* FlatChunkSource: A Superflat-style terrain generator.
* CheckerboardBiomeSource: A checkerboard-style biome generator that places plains and desert biomes in alternating chunks.
